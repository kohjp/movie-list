# 영화 뭐 볼까?

The Movie Database(TMDB) 에서 OPEN API를 통해 실시간 인기순으로 영화 리스트를 보여주는 웹 애플리케이션입니다.


TMDB : https://www.themoviedb.org/

배포 : https://movie-list-sigma-tawny.vercel.app/

언어 및 프레임워크 : TypeScript, Next.js
